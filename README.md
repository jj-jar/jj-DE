# jj-DE

This is the Waybar configuration stuff from my custom Linux From Scratch (LFS) build. 

The config file inside the waybar folder has some custom media buttons I tried to make for picking movies, tv shows, and music using foot terminal shortcuts and scripts that are there. I also tried to map a volume toggle switch for pulse audio. 

None of it really works right now. Clicking the buttons on the bar does not open the terminal windows or trigger the scripts, and the volume slider is out of sync. I kind of gave up on it for now.

If you are into LFS or know how to make Waybar launch bash scripts and terminal macros on Wayland properly, feel free to open an issue or leave some tips or cool things to add!
